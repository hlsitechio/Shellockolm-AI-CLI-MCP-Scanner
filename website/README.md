# Shellockolm Website

The official showcase website for Shellockolm - your security detective for React, Next.js, Node.js, and npm.

## Tech Stack

- **React 18** + **TypeScript**
- **Vite** for fast development
- **Tailwind CSS** with custom design tokens
- **shadcn/ui** components

## Development

```bash
cd website
npm install
npm run dev
```

## Build

```bash
npm run build
```
